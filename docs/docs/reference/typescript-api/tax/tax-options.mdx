### orderTaxCalculationStrategy

<MemberInfo kind="property" type={`<a href='/reference/typescript-api/tax/order-tax-calculation-strategy#ordertaxcalculationstrategy'>OrderTaxCalculationStrategy</a>`} default={`<a href='/reference/typescript-api/tax/order-tax-calculation-strategy#defaultordertaxcalculationstrategy'>DefaultOrderTaxCalculationStrategy</a>`}  since="3.6.0"  />

Defines how order-level tax totals and the tax summary are calculated.

The default strategy rounds tax at the individual line level and then sums
(per-line rounding). The [OrderLevelTaxCalculationStrategy](/reference/typescript-api/tax/order-tax-calculation-strategy#orderleveltaxcalculationstrategy) alternative
groups net subtotals by tax rate and rounds once per group (per-total rounding),
which is required by certain jurisdictions and ERP systems.
